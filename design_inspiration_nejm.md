# NEJM Design Reference

This project will use the current NEJM brand feel as inspiration, not as a literal reproduction.

## Core aesthetic

- Editorial and academic rather than startup-like
- White or very light gray canvas with sharp black typography
- Small number of strong accent colors, used sparingly
- Dense information presentation with disciplined hierarchy
- Quiet, authoritative visual tone

## Palette

Observed from NEJM-affiliated style materials:

- `#FF3300` NEJM red
- `#084E9B` link blue
- `#007AAB` secondary blue
- `#808285` dark gray
- `#EDF2F6` light gray
- `#FFFFFF` white
- `#000000` black

Recommended use in the CV:

- Use red for a small number of structural accents: active section markers, rules, current-position highlight, selected filters.
- Use blue for links, references, and interactive scholarly metadata.
- Keep the page mostly white or light gray with black text.
- Avoid broad saturated color fields.

## Typography

Official NEJM PDFs embed custom families including:

- `OTNEJMQuadraat`
- `OTNEJMScalaSansLF`
- `NEJMGalliard`
- `NEJMUnivers`

Design interpretation for this CV:

- Use a classic serif for display and longform scholarly cues.
- Use a restrained sans-serif for UI labels, navigation, metadata, dates, and controls.
- Favor small caps, uppercase labels, and tight letter spacing for section metadata.
- Keep body typography highly readable and not oversized.

Open web-safe substitute direction if the NEJM fonts are unavailable:

- Display serif: `Cormorant Garamond`, `EB Garamond`, or `Libre Baskerville`
- UI sans: `Source Sans 3`, `IBM Plex Sans`, or `Newsreader Sans` equivalent if already present

## Layout cues

- Thin rules and dividers instead of heavy cards
- Strong top matter and masthead treatment
- Multi-column editorial rhythm where appropriate on desktop
- Generous left-right whitespace around primary reading columns
- Minimal rounded corners; prefer square geometry
- Visual emphasis through type contrast, not decorative surfaces

## Interaction translation for the CV

- Navigation should feel like a journal index, not a dashboard
- Publication and contribution entries should read like citations first, cards second
- Filters and expandable details can be interactive, but the default state should remain calm and print-like
- Use motion lightly: fades, section reveals, and subtle line growth only

## Constraints

- The main `nejm.org` homepage blocks simple non-browser inspection, so these notes combine:
  - current NEJM-affiliated style guide colors
  - official NEJM PDF typography assets
  - publicly accessible NEJM brand materials
- Treat this as a design direction, not a brand recreation
